----------------------- Good to know about sqlite and JDBC ----------------
---------------------------------------------------------------------------

- Pour ajouter le JDBC Sqlite dans eclipse:
-------------------------------------------
clique droit sur le dossier du projet (racine)
properties
Java Build Path
Add External JARs
et ajouter le jar du jdbc téléchargé préalablement par exemple ici il s'appelle sqlite-jdbc-3.8.7.jar

- Pour ouvrir une base de données:
----------------------------------
public class SQLiteJDBC
{
  public static void main( String args[] )
  {
    Connection c = null;
    try {
      Class.forName("org.sqlite.JDBC");
	//le fichier se trouve dans le racine du projet
      c = DriverManager.getConnection("jdbc:sqlite:writingMentor.db"); 
    } catch ( Exception e ) {
      System.err.println( e.getClass().getName() + ": " + e.getMessage() );
      System.exit(0);
    }
    System.out.println("Opened database successfully");
  }
}

- Pour créer un dump d'une base de donnée et obtenir en sql les commandes utilisées pour la création:
ouvrir cmd.exe
naviguer dans le dossier ou il y a le exe de sqlite et le fichier .db de la base à dump
taper la commande:
sqlite3 nomdb.db
.output nomfichierdump.txt
.dump

- Pour récupérer un base à partir d'un dump
.read nomfichierdump.txt
